# Resistance_Calculator

This project shows:
Loops
Conditional statements
User input
Mathematical calculations
Electrical engineering concepts
Basic engineering software development


Key Features :
Input Handling: It asks for the number of resistors first, allowing for a flexible number of components rather than hardcoding just two.

Error Prevention: In the parallel function, it checks for zero-ohm inputs to prevent a "division by zero" error.

Looping Interface: The do-while loop in the main function keeps the program running so you can perform multiple calculations without restarting the app.
